# Interactive Transportation Network Application

A Java-based application demonstrating graph algorithms through a transportation network of cities and roads.

## Features

* Shortest path calculation using Dijkstra’s Algorithm
* BFS traversal of transportation network
* Reachability analysis
* Minimum Cost Network using Prim’s MST

## Technologies Used

* Java
* Graph Algorithms
* Swing GUI

## Algorithms Implemented

* Dijkstra’s Algorithm
* Breadth-First Search (BFS)
* Prim’s Minimum Spanning Tree (MST)

## Concepts Covered

* Graph Theory
* Data Structures
* Network Optimization
